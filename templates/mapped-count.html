Per Country

This map shows quantitative data on how distributed the tor network is. If you hover over a country, you can see the quantity of the selected data point. The darker green the country is, the higher of the respective count it has. Gray countries contain no nodes in that category..

This data is not normalized to the surface area of the countries. Normally in mapping this would be a bad idea, because it could mis-represent data. For these maps the goal is to demonstrate how many tor relays are in different government regions. So to normalize by area of the country would misrepresent the data.

Currently Intergvovernnental agencies and IPs which do not match to a country in GeoIP are not shown. They can be found in the raw data.

If you see a country under-represented, why not consider running a relay? It's fun, easy, and a great way to help privacy and anonymity on the Internet. More on running a relay can be found here Many people choose to run relays on a VPS, some providers can be found here.

This data was sourced from The Tor Project's Onionoo. Onionoo collects data from the tor network consensus and makes it easily retrievable. The raw data used to generate this map can be found here. This map is free open source software, if you find issues or have improvements let the authors know!
